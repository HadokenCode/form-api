
## Installation

### Requirements

- Docker 17.09.0-ce or above
- Docker Compose 1.16.1 or above
- Go 1.9.2 or above
- GNU Make 3.81 or above

### From source

```bash
$ go get -d -u github.com/kamilsk/form-api
$ cd $GOPATH/src/github.com/kamilsk/form-api
$ make deps test
```
